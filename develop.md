## 初始化
npm -> v6.1
electron -> v1.1

```
mkdir electron-works
cd electron-works

npm init

mkdir app
mkdir fonts
mkdir styles
mkdir scripts
mkdir images

touch index.html
```

## 基础库
```
npm install electron-prebuilt --save-dev
npm install electron-packager --save-dev
npm install electron-debug --save-dev
```
